## Installation

### From Pulsar Package Manager

```bash
ppm install latex-tools
```

### From Source

Clone this repository into your Pulsar packages directory:

```bash
cd ~/.pulsar/packages
git clone https://github.com/your-username/latex-tools.git
cd latex-tools
ppm install
```
